- Affine
- AMSCO
- Atbash
- Bifid
- Caesar
- Columnar
- Disrupted
- Double Columnar
- Four Square
- Playfair
- Polybius
- Rail-fence
- Route
- ROT13
- Substitution
- Trifid
- Turning Grille
- Two Square
